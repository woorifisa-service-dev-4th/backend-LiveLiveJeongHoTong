# 📢 생생정호통 (backend-LiveLiveJeongHoTong)
## 📌 프로젝트 소개
**생생정호통**은 강사와 학생이 실시간으로 채팅할 수 있는 콘솔 기반 채팅 애플리케이션입니다.


## 📌사용자 흐름: 채팅방 생성 및 입장
![image](https://github.com/user-attachments/assets/4fce815e-0a48-4dff-ae5d-50021f3bb6eb)


## 📌 프로젝트 구조
```plaintext
/backend-LiveLiveJeongHoTong
│── src/
│   ├── main/java/com/example/chat/
│   │   ├── executor/            # 실행 객체 
│   │   ├── model/               # 데이터 모델 (User, ChatRoom, Message)
│   │   ├── repository/           # 데이터 저장소 (채팅방 및 사용자 관리)
│   │   ├── service/              # 비즈니스 로직 (채팅방 및 로그인 관리)
│   │   ├── validator/            # 입력값 검증
│   │   ├── util/                 # 콘솔 출력 및 유틸리티
│   │   ├── Main.java  # 메인 실행 클래스
```
## 📌 주요 기능
### 🚀 Java Security 기반 로그인 시스템

Host로 로그인시 , Java Security를 활용한 ID & 비밀번호 검증 후 로그인 여부 확인

### 🚀 채팅방 관리

채팅방 생성 (방 이름, 방 번호, 생성일 저장)
생성된 채팅방 목록 조회
채팅방 입장 및 실시간 메시지 주고받기

### 🚀 입력값 검증

1 또는 2만 입력 가능 (잘못된 입력입니다. 1,2번 중 선택해주세요)
채팅방 입장 시 방 번호 입력 검증
